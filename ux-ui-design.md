# UX/UI Дизайн: Т-Командировка

> AI-First корпоративная платформа для командировок
> Версия: 1.0 | Дата: 28 февраля 2026

---

## 1. Дизайн-философия

### 1.1. Принцип: AI-First, Human-Friendly

Т-Командировка строится по принципу **«AI-first»** — главная точка входа в продукт это не формы и фильтры, а **разговорный AI-агент**, который понимает естественный язык. Пользователь описывает задачу как человеку — система делает всё остальное.

### 1.2. Дизайн-принципы

| # | Принцип | Описание | Пример |
|---|---------|----------|--------|
| 1 | **Conversation First** | Любое действие можно начать текстом в чате AI-агента | «Мне нужно в Казань во вторник на два дня» |
| 2 | **Zero Friction** | Минимум кликов от намерения до результата | Бронирование за 2 минуты, согласование за 1 tap |
| 3 | **Progressive Disclosure** | Простой интерфейс по умолчанию, сложность — по запросу | Базовый поиск → расширенные фильтры по клику |
| 4 | **Transparency** | Пользователь всегда видит, что происходит и почему | AI показывает источники цен, статус согласования |
| 5 | **Ecosystem Continuity** | Единый визуальный язык с Т-Банком, бесшовный переход | Из приложения Т-Банка → Т-Командировка без re-login |

#### Функциональные требования AI-агента

| ID | Требование | Детали |
|----|-----------|--------|
| **AI-UX-1** | **Typing Indicator** | Анимация «AI думает...» — три пульсирующие точки в стиле T-Blue. Появляется мгновенно после отправки сообщения. |
| **AI-UX-2** | **Streaming Response** | Текст AI появляется посимвольно (streaming), как в ChatGPT. Карточки билетов/отелей появляются блоками с плавной анимацией (fade-in + slide-up, 300ms). |
| **AI-UX-3** | **Inline Cards** | Результаты поиска отображаются как интерактивные карточки внутри чата. Кнопка «Выбрать» прямо на карточке. Максимум 3 варианта по умолчанию + «Показать ещё». |
| **AI-UX-4** | **Policy Badges** | Каждый вариант содержит badge тревел-политики: ✅ «В рамках политики» (зелёный) или ⚠️ «Превышает лимит на X ₽» (оранжевый). |
| **AI-UX-5** | **AI Recommendations** | Блок «💡 AI-совет» с голубым фоном (#F0F4FF). Предиктивные подсказки о ценах, персональные рекомендации на основе истории. |
| **AI-UX-6** | **Quick Reply Buttons** | После ответа AI — кнопки быстрых действий: «Забронировать всё», «Изменить параметры», «Добавить такси». |
| **AI-UX-7** | **Context Memory** | AI помнит контекст разговора. «Добавь такси из аэропорта» — AI понимает, что речь о Казани, о той же командировке. |
| **AI-UX-8** | **Source Transparency** | AI указывает источники цен: «Цена от S7 Airlines, актуальна на 10:30». По аналогии с Perplexity — прозрачность источников. |
| **AI-UX-9** | **Error Handling** | Если AI не понял: «Уточните, пожалуйста — вы хотите забронировать для себя или для сотрудника?». Не тупиковые ответы, а уточняющие вопросы. |
| **AI-UX-10** | **Multimodal Input** | Поддержка: текст + прикреплённое фото чека (OCR) + голосовой ввод (иконка микрофона). |
