## 1. Обзор продукта

### 1.1. Название

**Т-Командировка** — корпоративная платформа управления деловыми поездками и мероприятиями в экосистеме Т-Банка.

### 1.2. Видение (Vision)

Стать единственным сервисом на российском рынке, объединяющим три слоя в одном продукте:

- **Банковский слой** — зарплатный проект, корпоративные карты, кэшбэк, отсрочка, кредитные линии
- **Тревел-слой** — полный цикл от заявки до авансового отчёта + MICE-мероприятия
- **Lifestyle-слой** — такси, питание, каршеринг, заправки

### 1.3. Миссия

Избавить компании от хаоса командировок: когда сотрудник летит — ему удобно, когда компания платит — ей прозрачно, когда бухгалтер отчитывается — всё уже готово.

### 1.4. Целевые метрики (North Star)

| Метрика | Описание | Цель (12 мес.) | Цель (24 мес.) |
|---------|----------|---------------|----------------|
| **GMV** | Общий объём бронирований через платформу | 5 млрд ₽ | 15 млрд ₽ |
| **Активные компании** | Компании с ≥1 бронированием/мес. | 2 000 | 8 000 |
| **NPS** | Net Promoter Score | ≥ 55 | ≥ 65 |
| **Время бронирования** | Среднее время от заявки до подтверждения | < 5 мин. | < 2 мин. |
| **Экономия клиента** | Средняя экономия бюджета компании-клиента | 20% | 30% |

### 1.5. Ключевое ценностное предложение (UVP)

> **«Вся командировка — от приказа до авансового отчёта — в одном банке. Бронируйте, платите, отчитывайтесь — без бумаг, чеков и лишних сервисов.»**

### 3.2. Ролевая модель

| Роль | Доступ | Действия |
|------|--------|----------|
| **Сотрудник** | Мобильное приложение, Telegram-бот, веб | Создание заявки, бронирование, трекинг расходов, авансовый отчёт, личные поездки |
| **Руководитель** | Веб-кабинет, Telegram-бот, мобильное | Согласование заявок, установка лимитов, просмотр расходов подразделения |
| **Тревел-менеджер** | Веб-кабинет (расширенный) | Управление всеми поездками, настройка тревел-политики, групповые бронирования, MICE |
| **Бухгалтер** | Веб-кабинет, 1С-интеграция | Приём авансовых отчётов, закрывающие документы, НДС, выгрузка в 1С |
| **HR-менеджер** | Веб-кабинет | Приказы, служебные задания, КЭДО, организация мероприятий |
| **CFO / Администратор** | Веб-кабинет (admin) | BI-аналитика, бюджетирование, тревел-политика, управление ролями, настройки |

## 13. Определения и глоссарий

| Термин | Определение |
|--------|------------|
| **GDS** | Global Distribution System — система бронирования авиабилетов (Amadeus, Sirena, Sabre) |
| **MICE** | Meetings, Incentives, Conferences, Exhibitions — индустрия корпоративных мероприятий |
| **Тревел-политика** | Набор правил компании, определяющих допустимые расходы на командировки |
| **Авансовый отчёт (АО-1)** | Документ, которым сотрудник отчитывается о расходах в командировке |
| **Приказ Т-9** | Унифицированная форма приказа о направлении работника в командировку |
| **ЭДО** | Электронный документооборот — обмен юридически значимыми документами в электронном виде |
| **КЭДО** | Кадровый электронный документооборот |
| **NPS** | Net Promoter Score — индекс лояльности клиентов |
| **CSAT** | Customer Satisfaction Score — оценка удовлетворённости |
| **GMV** | Gross Merchandise Value — общий объём транзакций через платформу |
| **CAC** | Customer Acquisition Cost — стоимость привлечения клиента |
| **LTV** | Lifetime Value — совокупный доход от клиента за всё время сотрудничества |
| **OCR** | Optical Character Recognition — распознавание текста с изображений |
| **SSO** | Single Sign-On — единая точка входа через корпоративную учётную запись |
| **BFF** | Backend for Frontend — серверный слой, адаптирующий API под конкретный клиент (веб, мобильный) |
